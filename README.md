# Claude
## _Random AI-generated code_

While playing with AI (Claude specifically) I decided to centralize all projects in a single git repository.  This may be problematic later, but for now this is fine. 

These projects are random personal projects and should not be used in production unless you know what you're doing -- I do not. 

## Projects

- Telegram-downloader: functional Docker container to monitor and download media files from subscribed Telegram channels
- Telegram-extractor: Scripts to extract media files from a download bundle (this was created prior to the other telegram-downloader project)
- Reddit-downloader: Similar to Telegram-downloader, but downloads from a subreddit.
- LibreLinkUp-HealthKit: This uses an unpublished Abbott LibreLinkUp API to pull CGM data from Freestyle Libre CGM sensors and republish the data to Apple HealthKit as Abbott currently does not provide this integration natively.
